คุณคือ Senior Full-stack Software Architect, Business Analyst และ Project Manager

ฉันต้องการสร้าง Web Application ระบบร้องเรียนของศูนย์ดำรงธรรมจังหวัดศรีสะเกษ

Tech Stack ที่ต้องการใช้:
- Frontend: React 18 + Vite 5 + MUI 5
- Backend: Node.js 20 LTS + Express 4
- Database: MySQL 8
- Container (Dev): Docker Compose + phpMyAdmin
- Deploy: Railway (single-container จาก Dockerfile multi-stage — ไม่ใช้ Nginx แยก)

ข้อกำหนดสำคัญ:
- ตอนนี้อยู่ในช่วง Planning Only
- ห้ามเขียน code
- ห้ามสร้างไฟล์โปรแกรม
- ให้ช่วยวิเคราะห์ ออกแบบ และจัดทำแผนก่อนเท่านั้น
- ระบบต้องรองรับการรับเรื่องร้องเรียน การส่งต่อหน่วยงาน การติดตามสถานะ การแจ้งเตือน และ Dashboard

บริบทของระบบ:
ระบบนี้ใช้สำหรับศูนย์ดำรงธรรมจังหวัด เพื่อรับเรื่องร้องเรียนจากประชาชน บันทึกข้อมูลเรื่องร้องเรียน ส่งต่อไปยังหน่วยงานที่รับผิดชอบ ติดตามความคืบหน้า กำหนดระยะเวลาดำเนินการ และแสดงผลใน Dashboard สำหรับผู้บริหาร

ช่วยเริ่มจากการสรุปภาพรวมระบบ โดยยังไม่ต้องออกแบบ database และยังไม่ต้องเขียน code
