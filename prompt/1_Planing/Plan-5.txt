ช่วยออกแบบ MySQL Database Schema สำหรับระบบร้องเรียนศูนย์ดำรงธรรม

เงื่อนไข:
- ใช้ MySQL 8
- ตั้งชื่อตารางและ field เป็นภาษาอังกฤษแบบ snake_case
- ยังไม่ต้องเขียน SQL CREATE TABLE แบบสมบูรณ์
- ขอเป็น Database Design Overview ก่อน
- ต้องรองรับการขยายระบบในอนาคต
- ต้องมี audit log หรือประวัติการเปลี่ยนสถานะ

ให้แสดงผลเป็น:
1. รายชื่อตารางที่ควรมี
2. วัตถุประสงค์ของแต่ละตาราง
3. Field สำคัญของแต่ละตาราง
4. Primary Key
5. Foreign Key
6. ความสัมพันธ์ระหว่างตาราง
7. ตาราง master data ที่ควรมี
8. ตาราง transaction ที่ควรมี
9. ตาราง log/history ที่ควรมี
10. ข้อควรระวังเรื่องข้อมูลส่วนบุคคล
