ช่วยออกแบบ Dashboard, Report และ Notification สำหรับระบบร้องเรียนศูนย์ดำรงธรรม

ยังไม่ต้องเขียน code

Dashboard ต้องตอบคำถามผู้บริหาร เช่น:
1. มีเรื่องร้องเรียนทั้งหมดกี่เรื่อง
2. เรื่องใหม่วันนี้กี่เรื่อง
3. เรื่องอยู่ระหว่างดำเนินการกี่เรื่อง
4. เรื่องเกินกำหนดกี่เรื่อง
5. หน่วยงานใดมีเรื่องค้างมากที่สุด
6. ประเภทเรื่องใดพบมากที่สุด
7. พื้นที่ใดมีเรื่องร้องเรียนมากที่สุด
8. แนวโน้มรายเดือนเป็นอย่างไร

ให้แสดงผลเป็น:
- Dashboard Cards
- Charts
- Tables
- Filters
- Report Export
- Notification Rules
- SLA Alert Rules
- ตัวอย่างข้อมูลที่ใช้คำนวณแต่ละตัวชี้วัด
