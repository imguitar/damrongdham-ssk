จากแผนระบบทั้งหมด

ช่วยออกแบบ Project Structure และ Docker Architecture

Tech Stack:
- frontend: React 18 + Vite 5 + MUI 5
- backend: Node.js 20 LTS + Express 4
- database: MySQL 8
- phpMyAdmin (Dev)
- Docker Compose (Dev)
- Production: Railway (single-container จาก Dockerfile multi-stage — backend serve frontend dist ที่ ./public + /api; ไม่ใช้ Nginx)

ใช้ค่าจริงตามนี้ (ตรงกับ repo):
- โฟลเดอร์ DB ชื่อ `db/init/` (ไม่ใช่ database/) — mount เข้า /docker-entrypoint-initdb.d
- Backend entry: `backend/src/server.js`
- Ports (Dev host): frontend 5173, backend 5001, MySQL 3307→3306, phpMyAdmin 8081→80
- Prod รองรับ 2 targets (image เดียวกันจาก root `Dockerfile`): (A) Railway = `Dockerfile`+`railway.toml`; (B) On-premise = `docker-compose.prod.yml` + `nginx/default.conf`

ยังไม่ต้องเขียน code
