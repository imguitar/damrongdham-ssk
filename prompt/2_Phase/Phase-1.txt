งานที่ต้องการ:

วิเคราะห์ว่า Phase นี้ต้องสร้างหรือแก้ไขไฟล์ใดบ้าง
แสดงรายการไฟล์ที่จะสร้างหรือแก้ไขก่อน
เขียน code เฉพาะไฟล์ที่จำเป็นใน Phase นี้
แสดงโครงสร้างไฟล์หลังทำเสร็จ
อธิบายวิธีติดตั้ง dependency ถ้ามี
อธิบายวิธีรันระบบ
อธิบายวิธีทดสอบ
ระบุ Acceptance Criteria ของ Phase นี้
ระบุ TODO สำหรับ Phase ถัดไป โดยยังไม่ต้องทำ
แนะนำ Git commit message สำหรับ Phase นี้

เริ่มจากสรุปขอบเขตของ Phase

ต่อด้วย code แยกตามไฟล์
ต่อด้วยวิธีรัน
ต่อด้วยวิธีทดสอบ
ต่อด้วย Acceptance Criteria Checklist
ปิดท้ายด้วย Git commit message
