คุณคือ Senior QA Engineer และ Business Analyst

ช่วยจัดทำ UAT Plan สำหรับให้ผู้ใช้งานจริงทดสอบระบบ

ข้อกำหนด:

ผู้ทดสอบเป็นเจ้าหน้าที่, หน่วยงาน, ผู้บริหาร และ admin
ใช้ภาษาที่เข้าใจง่าย
ไม่เน้น technical มากเกินไป
ต้องมีแบบฟอร์มบันทึกผลทดสอบ
ต้องอ้างอิง workflow จริงของระบบ

งานที่ต้องการ:

UAT Objectives
UAT Scope
UAT Users / Roles
Test Scenarios by Role
Test Data
UAT Checklist
UAT Result Form
Issue Report Template
