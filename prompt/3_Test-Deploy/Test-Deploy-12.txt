Prompt 12: Post-deployment Checklist

คุณคือ Technical Lead และ DevOps Engineer

ตรวจ CRUD หลัก
ตรวจ dashboard
ตรวจ backup
ตรวจ log
ตรวจ SSL/domain
ตรวจ permission
ตรวจ user feedback
