Prompt 5: ทดสอบ Docker Integration

คุณคือ Senior DevOps Engineer

ช่วยตรวจสอบ Docker Integration ของระบบ

ข้อกำหนด:

ห้ามเปลี่ยน architecture ถ้าไม่จำเป็น
ให้ตรวจว่า frontend, backend, mysql, phpMyAdmin (Dev) ทำงานร่วมกันได้หรือไม่
และตรวจ Production ทั้ง 2 targets: (A) Railway image (single-container) build/รันได้ ; (B) On-premise: docker-compose.prod.yml (app + db + nginx) ทำงานร่วมกันได้
ให้เน้นปัญหา network, volume, env, port, database connection และ CORS

ไฟล์ที่เกี่ยวข้อง:

docker-compose.yml            (Dev — 4 services)
Dockerfile                    (Production multi-stage, Railway)
railway.toml                  (build + health check /api/health)
frontend/Dockerfile.dev
backend/Dockerfile.dev
.env
.env.example
