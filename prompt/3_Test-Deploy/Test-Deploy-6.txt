Prompt 6: Security Review เบื้องต้น

คุณคือ Senior Application Security Reviewer

ช่วยตรวจสอบความปลอดภัยเบื้องต้นของ Web Application ก่อน deploy

ข้อกำหนด:

ห้ามเพิ่ม feature ใหม่

ให้แนะนำวิธีแก้ที่ทำได้จริง

Authorization

File upload

Rate limiting
HTTPS / reverse proxy
Database access
Backup security
Logging and audit log
